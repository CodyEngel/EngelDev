This is the first post on my new blog. I recently rebuilt my personal site using [Eleventy](https://www.11ty.dev/), a simpler static site generator that gives me full control over the output.

## Why Eleventy?

I wanted a static site generator that stays out of the way. Eleventy lets me write content in Markdown, use Liquid templates, and ship plain HTML and CSS without any client-side JavaScript framework overhead.
